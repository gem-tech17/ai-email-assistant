# 📧 AI Email → Action Assistant

An AI-powered tool that analyzes emails, extracts actionable tasks, and generates professional responses.

## 🚀 Features
- 📌 Email summarization
- 🎯 Intent detection
- ⚡ Priority classification
- ✅ Task extraction
- ✉️ Automated reply generation

## 🛠 Tech Stack
- Python
- Streamlit
- Ollama (Local LLM)

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

## 💡 Example

Input:
"Schedule a meeting tomorrow and send the latest report"

Output:
- Summary
- Intent
- Tasks
- Suggested Reply
